# Video poster images (optional)

Add a still image here for each project to show before its video plays.
Reference it via the `poster` field in `src/data/projects.ts`
(e.g. `/posters/fittrack.jpg`).

- Match the video's aspect ratio (9:16 for mobile, 16:9 for web).
- A good poster is usually the most visually appealing frame of the demo.
